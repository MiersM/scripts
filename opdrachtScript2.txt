Schrijf een script, dat steeds het volgende herhaalt tot er op "ctrl+c" wordt gedrukt:
- CPU en geheugen info, alsook informatie over de hostnaam, het IP adres en de default gateway van de computer, wordt na elkaar weergegeven
- beperk de schermoutput, zodat enkel op een lijn het CPU type weergegeven wordt, en op een andere lijn het totale beschikbare geheugen, en op de laatste lijn het IP adres + GW
- laat aan deze informatie, met echo, een logische vermelding van de inhoud voorgaan, bvb "Je IP adres is 192.168.0.1"
- geef ten slotte ook de procentuele verhouding weer tussen het gebruikte geheugen, en het totale beschikbare geheugen. 
- Deze informatie wordt voorafgegaan door een lijntje "<student> zegt je dat ... :", waar <student> een lijst van parameters is, die op de command line kan meegegeven worden met het script.

Het inlezen van "een lijst parameters na elkaar", kan met behulp van het shift commando (schuif een parameter op), zoals in ...

until [ er zijn geen parameters meer ]; do
  lijst="$lijst $1"
  shift
done
